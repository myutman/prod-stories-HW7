# prod-stories-HW7

## Запуск бота
* Создать окружение
<pre>$ conda env create -f environment.yml</pre>
* Установить зависимости
<pre>$ pip3 install -r requirements.txt</pre>
* Запустить бота
<pre>$ python3 bot.py</pre>
